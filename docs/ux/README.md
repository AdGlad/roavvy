## Design Principles

1. **Discovery over data entry.** The app should surface what the user has already done, not ask them to log it.
2. **Privacy as a feature.** The fact that photos never leave the device is a selling point — design the onboarding to communicate this clearly.
3. **Offline as normal.** Offline states should feel seamless, not broken.
4. **Earned delight.** Motion and celebration (achievements, map reveals) should feel meaningful, not noisy.
