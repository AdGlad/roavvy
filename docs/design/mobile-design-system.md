# Roavvy Mobile Design System

## Product Vision

Roavvy should feel:

- adventurous
- premium
- energetic
- youthful
- stylish
- emotionally rewarding
- visually exciting
- modern mobile-first

The experience should feel closer to:

- a premium travel social app
- an Apple-quality lifestyle product
- a modern creative editing app
- a travel memory experience

NOT:

- enterprise software
- a settings-heavy utility
- a generic ecommerce app
- a cluttered social feed

---

# Core Emotional Themes

Roavvy is about:

- memories
- discovery
- movement
- identity
- achievement
- storytelling
- travel pride

The app should create emotional moments:

```text
"I've actually done all this."
"I want to share this."
"I want to wear this."
```

# Design Principles

## 1. Preview First

Always prioritise visual content over controls.

Examples:

* shirt previews stay visible
* map stays dominant
* passport stamps are hero content
* cards are visual first

Avoid:

* long forms
* hidden imagery
* giant settings panels

---

## 2. Compact Controls

Controls should feel:

* lightweight
* touch-friendly
* visually obvious
* modern

Use:

* segmented controls
* chips
* sliders
* icon buttons
* bottom sheets
* floating overlays

Avoid:

* stacked dropdown forms
* oversized settings rows
* excessive scrolling

---

## 3. Mobile-First Layouts

All experiences must feel designed specifically for phones.

Requirements:

* thumb-friendly
* minimal typing
* compact interaction zones
* highly visual navigation

Avoid:

* desktop-style layouts
* excessive menus
* multi-column complexity

---

## 4. Delight and Energy

Roavvy should feel alive.

Use:

* subtle motion
* animated transitions
* glow
* confetti
* celebration pulses
* responsive interactions

But:

* avoid overwhelming animation
* avoid gimmicks
* keep interactions smooth and premium

---

# Buttons

Buttons should feel:

* tactile
* modern
* energetic

Use:

* rounded corners
* soft gradients
* glow/highlight states
* subtle motion on tap

Avoid:

---

# Chips and Segmented Controls

Preferred for:

* filters
* year selection

* placements

* swipeable
* modern

# Maps

The map is a hero feature.

Requirements:

* immersive
* cinematic
* smooth zooms
* glowing visited countries

Visited countries should feel rewarding.

Use:

* gold highlights
* subtle glow
* animated transitions

Avoid:

* static GIS-style visuals
* cluttered overlays

---

# Passport Stamp Styling

Passport stamps should feel:

* authentic
* layered
* collected over time
* tactile

Use:

* slight overlap
* adaptive scaling
* realistic stamp textures
* varied rotation

Avoid:

* rigid grids
* excessive overlap
* tiny unreadable stamps

---

# Merch Design Principles

Roavvy merch is:

* emotional memorabilia
* wearable memories
* identity expression

NOT:

* generic printed apparel

---

## Merch Preview Rules

Always:

* keep shirt preview visible
* prioritise visual design
* minimise settings clutter

---

## Merch Layout Rules

Preferred:

* portrait orientation first
* artwork fills available space
* controlled overlap only

Avoid:

* tiny artwork
* excessive whitespace
* chaotic composition

---

## Merch Text Rules

Titles:

* describe the memory/trip
* never mention shirt type

Good:

```text
Europe 2026
Island Memories
Across Three Continents
```

Bad:

```text
Passport Shirt
Flag Tee
Word Cloud Design
```

Subtitles:

* always start with:

```text
Roavvy:
```

---

# Animation Guidelines

Use:

* smooth easing
* subtle scale transitions
* globe movement
* animated pulses
* celebratory reveals

Avoid:

* jarring movement
* bouncing everything
* long delays

Animation should feel:

* elegant
* rewarding
* fluid

---

# Bottom Sheets

Bottom sheets are preferred over full settings pages.

Should:

* feel layered
* support drag interaction
* maintain context visibility

Use:

* rounded corners
* blurred background
* compact controls

---

# Achievement Experience

Achievements should feel exciting.

Use:

* confetti
* globe zooms
* glow
* cinematic reveal

The user should feel rewarded.

---

# Memory Pulse Experience

Memory Pulses should feel:

* nostalgic
* cinematic
* surprising

They should trigger emotional reflection and merch opportunities.

---

# Ecommerce Experience

Roavvy is NOT a traditional store.

Purchasing should feel like:

```text
Turning memories into physical artifacts
```

NOT:

```text
Browsing products in a shop
```

Merch should emerge naturally from:

* achievements
* memories
* travel moments

---

# UX Rules

## Always Prefer

* visual selection
* direct manipulation
* swipe interactions
* compact layouts
* previews
* progressive disclosure

---

## Avoid

* giant forms
* admin-style settings
* clutter
* hidden previews
* long scrolling pages
* duplicated controls

---

# Engineering Guidance for Claude

Before redesigning any screen:

1. Keep previews visible
2. Reduce vertical scrolling
3. Use visual selectors where possible
4. Prioritise emotional impact
5. Preserve existing working purchase logic
6. Reuse shared title/subtitle generation
7. Reuse shared colour logic
8. Maintain a premium modern aesthetic

---
