## Goal

Every achievement currently generates options using only the four existing templates (Passport, Flags, Heart Flags, Tour Dates). M103 adds two new renderable template types — Typography and Badge — and improves option curation so each achievement type generates a more varied, emotionally engaging set of shirt ideas.

The user should feel: "I actually want to wear this" — not "another autogenerated travel shirt."

## New Templates

### TypographyCard
A stacked country-names text composition.

- Lists up to 20 country names in two columns, alternating font size (larger/smaller) and opacity (100%/60%) for visual rhythm
- Smallest legible size: ~11sp; largest: ~18sp
- Dark navy background (#0D1B2A) with a subtle vertical gradient overlay
- Bottom-right: small "N countries" label in white/40%
- Transparent canvas border (same as grid/heart)
- Scales well for 1–50+ countries; single country uses the country name in large centred headline style

### BadgeCard
A circular explorer badge / achievement seal.

- Outer decorative ring: tick marks at regular intervals, thin stroke
- Inner area: up to 12 flag thumbnails arranged in an arc or ring formation; flags beyond 12 replaced by "+N more" text
- Central text: achievement scope label (e.g. "Europe Explorer", "10 Countries")
- Outer ring text: "ROAVVY · TRAVEL · [YEAR]" letterpress style
- Square canvas (1:1 aspect ratio), rendered transparently
- For a single country: full-width flag centred inside badge ring

## BadgeCard Aspect Ratio

`BadgeCard` uses a 1:1 (square) aspect ratio.

Add `CardTemplateType.badge` to `merchBackCardAspectRatio()` in `merch_option_list_widgets.dart` returning `1.0`.
`TypographyCard` uses the existing 3:2 landscape aspect ratio (same as grid/timeline).

| Risk | Mitigation |
|---|---|
| `BadgeCard` SVG flag loads time out in `CardImageRenderer` | Follow `HeartFlagsCard.onAssetsLoaded` pattern exactly; `assetsTimeout` defaults to 10s |
| `card_editor_screen.dart` switch cases for new templates need full editor support | Stub cases fallthrough to grid behaviour for now; full editor support deferred |
| Typography with 50+ country names is visually crowded | Truncate at 24 items with a "+ N more" label; cap font size minimum at 10sp |
| Badge at 1:1 aspect ratio may look odd in the 3:2 card list thumbnails | `MerchOptionCard` thumbnail already clips to a fixed height; square artwork will letter-box naturally |
