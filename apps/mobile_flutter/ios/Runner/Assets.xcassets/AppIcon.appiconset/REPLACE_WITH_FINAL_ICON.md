# App Icon — Awaiting Final Design

The PNG files in this directory are Flutter placeholder icons.

**Before App Store submission:**

1. Obtain the final Roavvy icon as a 1024×1024 PNG (no transparency, no rounded corners — iOS applies rounding automatically).
2. Use a tool such as [appicon.co](https://www.appicon.co) or Xcode's asset importer to generate all required sizes from the master 1024×1024 file.
3. Replace all `Icon-App-*.png` files in this directory with the generated assets.
4. Verify `Contents.json` references match the filenames.
5. Delete this file.

**Required sizes (iOS):** 20pt @1x/2x/3x, 29pt @1x/2x/3x, 40pt @1x/2x/3x, 60pt @2x/3x, 76pt @1x/2x, 83.5pt @2x, 1024pt @1x (App Store).
